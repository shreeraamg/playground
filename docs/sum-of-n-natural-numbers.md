# Sum of N natural numbers

$$Sum = 1 + 2 + 3 + .... + (n-1) + n$$

This can also be written as

$$Sum = n + (n-1) + (n-2) + .... + 2 + 1$$

Therefore, 2 times Sum will be

$$2 * Sum = [1 + 2 + 3 + .... + (n-1) + n] + [n + (n-1) + (n-2) + .... + 2 + 1]$$

$$2 * Sum  = (1 + n) + (2 + n - 1) + (3 + n - 2) + .... + (n - 1 + 2) + (n + 1)$$

$$2 * Sum = (1 + n) + (n + 1) + (n + 1) + .... + (n + 1) + (n + 1)$$

$$2 * Sum = n(n+1)$$

$$Sum = \frac{n(n+1)}{2}$$
